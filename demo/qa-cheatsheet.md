# Judge Q&A Cheatsheet

**Why voice first?**
RMs are often between meetings or in transit. Calling SYNC is faster than opening and reading CRM records.

**Why Ringg AI?**
Ringg handles the inbound/outbound voice agent layer, custom variables, phone numbers, and call events. SYNC supplies the banking intelligence and CRM orchestration.

**Is this real CRM-ready?**
Yes. The mock adapter is the demo default, and the HubSpot/Salesforce adapters show the concrete mapping points for contacts, deals/opportunities, tickets/cases, and activity logging.

**What happens without API keys?**
The local app runs in simulation mode so judges can see the full dashboard flow without external credentials.
